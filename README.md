# Landing page for astrology services

Статичный лендинг для продажи услуг астролога Кирилла. В проекте используется чистый HTML, Tailwind CSS (CDN) и JavaScript без дополнительных сборщиков.

## Структура
- `index.html` — основной файл лендинга
- Tailwind CSS подключён через CDN и отвечает за сетки, отступы и адаптивные состояния
- `scripts/main.js` — переключение тем, мультиязычность, раскрывающиеся описания, прелоадер, GSAP-анимации и отправка формы на Formspree

## Быстрый старт
Откройте `index.html` в браузере.

## Особенности
- автоматическое определение языка и темы + ручной переключатель языков
- SEO-метаданные, OG/Twitter-превью и структурированные данные обновляются вместе с языком
- подробные описания услуг открываются по кнопкам «Подробнее»/«Read more», итоговая стоимость доп. блоков считается сразу
- итоговая сумма и состав консультации подсвечиваются отдельным блоком и обновляются в реальном времени даже на мобильных устройствах
- прелоадер, параллакс и плавные анимации на GSAP/ScrollTrigger, а также Tailwind-классы для сеток и адаптива
- форма записи отправляет данные на Formspree (`https://formspree.io/f/mzzyaopy`) без перезагрузки страницы и показывает модальное подтверждение
