# Landing page for astrology services

Статичный лендинг для продажи услуг астролога Кирилла. В проекте используется чистый HTML, CSS и JavaScript без дополнительных сборщиков.

## Структура
- `index.html` — основной файл лендинга
- `styles/main.css` — стили оформления
- `scripts/main.js` — переключение тем, мультиязычность, раскрывающиеся описания и отправка формы на Formspree

## Быстрый старт
Откройте `index.html` в браузере.

## Особенности
- автоматическое определение языка и темы + ручной переключатель языков
- SEO-метаданные и структурированные данные обновляются вместе с языком
- подробные описания услуг открываются по кнопкам «Подробнее»/«Read more»
- форма записи отправляет данные на Formspree (`https://formspree.io/f/mzzyaopy`) без перезагрузки страницы
